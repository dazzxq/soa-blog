Create `services/search-service/src/Controllers/SearchController.php` (namespace `App\Controllers`, use `App\Db`, `App\Json`, `App\DomainError`, `PDO`, PSR-7 Request/Response).

`search(Request $req, Response $res): Response` — implement exactly per RESEARCH Pattern 1:
- `$q = trim((string)($req->getQueryParams()['q'] ?? ''))`; if `''` → `Json::list($res, [], ['total'=>0,'q'=>''])`.
- if `mb_strlen($q) > 100` → `throw new DomainError(400,'VALIDATION_FAILED','Từ khoá tìm kiếm quá dài.')`.
- `$limit = min(50, max(1, (int)($req->getQueryParams()['limit'] ?? 20)))`.
- `$term = '%' . $q . '%'` (bound as a VALUE — wildcards we add are literal SQL; user text fully driver-escaped; NEVER interpolate `$q`).
- SQL selects `user_id, username, display_name, headline, location, skills_text, avatar_url FROM search_index WHERE display_name LIKE :t1 OR username LIKE :t2 OR headline LIKE :t3 OR skills_text LIKE :t4 ORDER BY display_name ASC LIMIT :lim`.
- bind `:t1..:t4` = `$term` (4 distinct names — native prepared statements forbid reuse), bind `:lim` with `PDO::PARAM_INT`.
- map rows: cast `user_id` to int; return `Json::list($res, $rows, ['total'=>count($rows), 'q'=>$q])`.

`upsert(Request $req, Response $res): Response`:
- `$b = (array)($req->getParsedBody() ?? [])`; `$uid = (int)($b['user_id'] ?? 0)`; if `$uid <= 0` → DomainError(400, VALIDATION_FAILED, 'Thiếu user_id.').
- `INSERT INTO search_index (user_id, username, display_name, headline, location, skills_text, avatar_url) VALUES (:uid,:un,:dn,:hl,:loc,:sk,:av) ON DUPLICATE KEY UPDATE username=VALUES(username), display_name=VALUES(display_name), headline=VALUES(headline), location=VALUES(location), skills_text=VALUES(skills_text), avatar_url=VALUES(avatar_url)`.
- bind from body with null-coalescing (`username`/`display_name` default `''`; headline/location/skills_text/avatar_url default null).
- return `Json::ok($res, ['user_id'=>$uid], 200)`.
